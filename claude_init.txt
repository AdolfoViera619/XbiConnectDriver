---------------------------------------------------------
 ECOSISTEMA XbiConnect
---------------------------------------------------------
Tres clientes consumen el mismo backend Chatwoot v4.8.0:

  1. Android Agents      → Android/XbiConnect/
  2. iOS Agents          → Ios/XbiConnect/
  3. Android Driver      → Android/XbiConnectDriver/   ← ESTA SESIÓN

Drivers vs Agents:
  - Driver  = conductores (logística, tracking, viajes).
  - Agents  = customer support (gestión de tickets).
  - Ambos comparten backend, auth (JWT/headers + Keystore),
    modelos (Conversation, Message, User) y WebSocket.
  - Driver tiene namespace propio: `xbiconnect.android.driver`.

---------------------------------------------------------
 ARQUITECTURA DEL FEATURE driver_channel
---------------------------------------------------------
AUTH Y MODELO DE USUARIO
- Drivers son Chatwoot Users (mismo tipo que agentes/admins).
  NO son Contacts. Login = devise_token_auth (access-token/uid/client).
- Cada driver tiene su User, sus credenciales, su account membership.
- Mismo flujo de auth que el agent app — copiar AuthRepository.

INBOX TYPE
- Server agregó columna `driver_channel: boolean` en tabla inboxes.
- Inboxes para drivers: Channel::Api con driver_channel=true.
- API expone el flag en `/api/v1/inboxes` y dentro de
  `meta.channel` de conversaciones.
- Filtro: `inbox.driver_channel == true` para listar canales relevantes.

ANUNCIOS MASIVOS (BROADCAST)
- Usar Chatwoot Campaigns (no broadcast custom).
- Endpoint: /api/v1/accounts/{id}/campaigns
- Tipo: one_off para anuncios puntuales, ongoing para automáticos.

UI EN AGENT APPS (referencia)
- Cuando inbox.driver_channel=true, los agent apps muestran un
  icono de camión con fondo azul #1E40AF en lugar del icono de
  canal normal.
- En Driver app esto NO aplica (el driver solo ve sus propias
  conversaciones).

DEV: TOGGLE DE SERVIDOR LOCAL
- App debe tener un toggle "Use local server" en settings de
  desarrollo que apunte al docker dev del usuario.
- Producción: usar URL de XBI API (igual que agent app).

ALCANCE INICIAL DEL DRIVER APP (sin ETA/HOS por ahora)
- Login con email/password (Chatwoot User).
- Ver inboxes con driver_channel=true.
- Ver/enviar mensajes en conversaciones de esos inboxes.
- Ver anuncios (Campaigns) destinados al driver.
- ETA y HOS: deferred — se trabajan después.
